== Description ==

Creates a shortcode to automaticaly incorporate legal info in Wordpress-based websites. Requires Wordpress Plugin Framework Reloaded.

== Installation ==

How to install the Website Legal Info plugin and get it working in two easy steps:

1. Upload `/website-legal-info/` folder and its files to the `/wp-content/plugins/` directory or just use the Wordpress admin interface
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new page and insert [legal-info] in it. 
4. Save the page and that is all.

Also, check out the other [Wordpress plugins](http://wordpress.org/extend/plugins/profile/ipublicis) by the same author.

== Translations ==

If you want to make a translation for your language, use the legal-info.pot included and (if you want) send me the files (just the .po and .mo ones) to dev@ipublicis.com for inclusion in plugin's package. You'll be credited, of course.
